## Present

```bash
make serve
# Open http://localhost:8000/talks/gitops-at-scale/
```

Or open any `talks/*/index.html` directly in a browser.

## Tech Stack

- [reveal.js 5.1.0](https://revealjs.com/) via CDN (no npm install needed)
- Custom dark theme with Inter font
- Syntax highlighting via highlight.js
- Speaker notes support (press `S` while presenting)

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` / `Arrow Right` | Next slide |
| `Arrow Left` | Previous slide |
| `S` | Speaker notes view |
| `F` | Fullscreen |
| `Esc` | Overview mode |
| `B` | Blackout screen |
